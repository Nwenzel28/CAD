Learning Modules

Structured lessons designed for intermediate CAD users. Each module builds on the last — start at Module 01 or jump to whatever's most relevant to your project.

Module 01 Intermediate
Design for Manufacturing

Learn to design parts that can actually be built. Understand tolerancing, feature minimization, and how your software decisions translate to machine time and cost.

GD&T Basics Tolerancing DFM Principles Material Selection Draft Angles
~3 hrs 6 sections
Module 02 Intermediate
Iteration Workflows

Design is never one-and-done. Build systematic habits for versioning, documenting decisions, and using feedback cycles to improve designs quickly and intentionally.

Version Control Changelog Practice Rapid Prototyping Feedback Loops
~2.5 hrs 5 sections
Module 03 Intermediate
Assembly Design

Go beyond individual parts. Learn top-down design strategy, how to manage complex assemblies, and how design choices in one part cascade through an entire system.

Mates & Constraints Top-Down Design Assembly BOM Sub-Assemblies Clearance Analysis
~4 hrs 7 sections
Module 04 Advanced
Structural Analysis Basics

Get a working understanding of FEA tools within your CAD environment. Learn to identify load paths, set up basic simulations, and validate designs before building.

FEA Setup Load Path Analysis Stress Concentrations Factor of Safety
~3.5 hrs 6 sections
Module 05 Intermediate
Fasteners & Joining

The details that hold everything together — literally. Deep dive into fastener selection, thread engagement, press fits, adhesives, and when each joining method is appropriate.

Fastener Selection Thread Engagement Press Fits Weld Design
~2 hrs 4 sections
Module 06 Advanced
System-Level Thinking

Shift from part-level to system-level design. Learn how mechanical, electrical, and software constraints interact, and how to design for integration from the start.

Interface Control Cable Routing Mechatronics Basics Design Reviews
